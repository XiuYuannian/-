# 纸上回声：近年读书记

这是一个关于阅读与回望的项目。

本项目主要用于记录近些年阅读过的书籍、读后感与个人思考。这里的文字并不追求标准答案，也不试图对某本书作出权威评价，而更像是一份个人阅读档案：记录某一阶段的我，与某一本书相遇时产生的感受、疑问和思考。
